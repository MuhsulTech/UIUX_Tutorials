About This Course

This course teaches you everything you need to know about UI/UX design — from foundations to advanced concepts — with a strong focus on frontend implementation.

What you will learn
  • ◎ UX research & personas
  • ◎ Wireframing & prototyping
  • ◎ Visual UI design
  • ◎ Responsive frontend UI
Who this is for
  • ◆ Frontend developers
  • ◆ UI/UX beginners
  • ◆ Designers learning code
  • ◆ Anyone building websites
